# Snake Game

A basic Snake game made using **HTML**, **CSS**, and **JavaScript**.

## Features
- Snake movement with keyboard controls
- Food spawning
- Score tracking
- Game over on wall/self collision

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API (optional)

## Purpose
Learning project to understand frontend basics and game logic.
